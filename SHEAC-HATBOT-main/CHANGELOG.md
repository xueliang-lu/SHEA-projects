# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

---

## [2.0.0] - 2026-02-08

### Added
- Dual AI provider support (OpenAI GPT-4o-mini and Google Gemini 2.0 Flash)
- RAG system for FAQ knowledge base queries
- Full chat history with MongoDB persistence
- Modern landing page with animated UI
- Provider switching in chat interface
- TypeScript support throughout

### Changed
- Upgraded to Next.js 16.1.6
- Upgraded to React 19
- Upgraded to Tailwind CSS 4
- Migrated to App Router architecture

### Improved
- Enhanced error handling for AI API failures
- Better conversation management
- Responsive design for all screen sizes

---
